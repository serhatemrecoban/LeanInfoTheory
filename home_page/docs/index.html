Current Lean modules and build status.

Home Blueprint GitHub

Generated API documentation is the next documentation milestone. For now, this page records the compileable Lean files in the current foundation layer.

  • LeanInfoTheory.Basic: lightweight project namespace and shared status vocabulary.
  • LeanInfoTheory.Probability.Finite: real-mass bridge lemmas for finite PMF sums.
  • LeanInfoTheory.Shannon.Entropy: finite Shannon entropy and random-variable entropy.
  • LeanInfoTheory.Shannon.InfoMeasures: marginals, conditional entropy, mutual information, conditional mutual information, and support/orientation lemmas.
  • LeanInfoTheory.Shannon.SemanticBridge: separated home for future KL and conditional-law bridge theorems.
  • LeanInfoTheory.InformationMeasures: public re-export for the finite information-measure API.
  • LeanInfoTheory.EntropyExpr: formal entropy-expression algebra.
  • LeanInfoTheory.Certificate: certificate soundness theorem.
  • LeanInfoTheory.Examples: closed toy example.

Human-facing design notes are tracked in docs/foundation-conventions.md and docs/project-log.md in the repository.
