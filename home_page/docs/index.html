Module list

Current compileable Lean modules; generated API docs are planned.

Home Blueprint GitHub

This page is a hand-written module list, not generated declaration documentation. Generated API documentation is a later milestone.

  • LeanInfoTheory.Basic: lightweight project namespace and shared status vocabulary.
  • LeanInfoTheory.Probability.Finite: real-mass bridge lemmas and pointwise PMF.map facts for finite PMF sums and relabeling arguments.
  • LeanInfoTheory.Shannon.Entropy: finite Shannon entropy, random-variable entropy, and relabeling, swap, and reassociation invariance theorems.
  • LeanInfoTheory.Shannon.EntropyBounds: Jensen-based finite entropy upper bound entropy_le_log_card and uniform-law equality theorem entropy_uniformOfFintype. Import this module explicitly when those heavier analytic bounds are needed.
  • LeanInfoTheory.Shannon.InfoMeasures: marginals, conditional entropy, mutual information, conditional mutual information, and support/orientation lemmas.
  • LeanInfoTheory.Shannon.SemanticBridge: separated bridge layer containing selfInfo and entropy_eq_integral_selfInfo, with future KL and conditional-law bridge theorems planned here.
  • LeanInfoTheory.InformationMeasures: public convenience re-export for the finite information-measure API; theorem-oriented code should usually prefer the canonical LeanInfoTheory.Shannon names.
  • LeanInfoTheory.EntropyExpr: formal entropy-expression algebra and the empty-entropy convention interface.
  • LeanInfoTheory.EntropyVal: abstract Shannon entropy valuations for certificate work.
  • LeanInfoTheory.PrimitiveIneq: primitive Shannon inequality expressions and their soundness under abstract valuations.
  • LeanInfoTheory.Certificate: generic nonnegative-combination soundness theorem and exact decomposition matching.
  • LeanInfoTheory.Certificate.Checked: raw certificate data, a first raw-to-checked validator, and proof-carrying checked certificate structures with nonnegative rational coefficients and exact decomposition equality.
  • LeanInfoTheory.Certificate.Submodularity: separately importable first non-toy certificate demo, proving entropy submodularity from a validated CMI certificate.
  • LeanInfoTheory.Examples: separately importable closed toy examples for the certificate layers.

Human-facing design notes are tracked in docs/foundation-conventions.md and docs/project-log.md in the repository.
