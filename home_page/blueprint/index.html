LeanInfoTheory Blueprint

Project layers and theorem-dependency plan.

Home Documentation Dependency graph GitHub

This blueprint mixes implemented modules with planned layers. Planned items are marked as future work; the current Lean code is still an early foundation, not a completed network-information-theory library.

Layer 0: Existing foundations

  • mathlib binary and q-ary entropy
  • mathlib KL divergence and KL chain-rule infrastructure
  • mathlib PMFs, finite measures, Markov kernels, and coding foundations
  • PFR-style entropy and mutual-information infrastructure to audit

Layer 1: Finite information measures

  • finite random variables and mathlib PMFs
  • finite Shannon entropy in nats via Real.negMulLog
  • entropy invariance under equivalence, injective alphabet relabeling, swaps, and reassociation
  • finite entropy upper bound by log |alphabet| and the uniform-law equality case
  • joint entropy, conditional entropy, mutual information, and conditional mutual information
  • marginal mass formulas, support/domination lemmas, and coordinate-orientation lemmas
  • semantic bridge theorem identifying entropy with expected self-information over PMF.toMeasure
  • future semantic bridge theorems for conditional laws and KL divergence

Layer 2: Algebraic entropy expressions

  • entropy atoms indexed by finite sets of variable names
  • rational linear combinations of atoms
  • semantic evaluation into real entropy values
  • abstract Shannon entropy valuations for certificate soundness
  • primitive Shannon inequality expressions and valuation soundness theorems

Layer 3: Certificates

  • current nonnegative-combination soundness skeleton
  • current raw/checked certificate split with nonnegative rational coefficients
  • current exact rational expression matching for checked decompositions
  • current basic raw-to-checked validation for primitive-tagged steps
  • current entropy submodularity demo from a validated CMI certificate
  • future import support for external certificate formats
  • future import path for PSITIP/oXitip-style certificates

Layer 4: Network information theory

  • future data-processing examples
  • future Fano-style converse skeletons
  • future cut-set and multi-terminal converse examples
