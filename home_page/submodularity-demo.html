Submodularity Demo

The first checked entropy-inequality certificate in the project.

The demo proves the Shannon submodularity inequality through the same certificate-checking path that later examples should use.

Target Inequality

For entropy atoms A and B, the checked target is

H(A) + H(B) - H(A union B) - H(A inter B) >= 0.

Certificate Idea

Let C = A inter B, A' = A \ B, and B' = B \ A. The normalized expression above is exactly the conditional mutual information primitive

I(A';B' | C) = H(A',C) + H(B',C) - H(A',B',C) - H(C).

Since conditional mutual information is one of the primitive nonnegative Shannon inequalities in the abstract valuation layer, the one-step certificate proves the target inequality.

Proved demo Entropy submodularity

Lean proves H(A) + H(B) - H(A union B) - H(A inter B) >= 0.

  1. Build the formal entropy expression for submodularity.
  2. Represent the certificate as raw data with one CMI primitive step.
  3. Run the raw-to-checked validator.
  4. Check that the normalized rational expression equals the target.
  5. Use the generic certificate soundness theorem to obtain the inequality.

Current Limits

The project checks a supplied certificate, but it does not yet search for certificates automatically. Independence, Markov, and functional dependence assumptions are also future extensions of the certificate language.
