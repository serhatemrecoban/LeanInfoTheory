LeanInfoTheory is an early-stage Lean/mathlib project for finite information measures, entropy-expression certificates, and network-information-theory converse automation.

Current status

  • Lake project initialized with Lean 4.30.0 and mathlib 4.30.0.
  • Mathlib cache fetched and the project builds locally.
  • Initial Lean modules define the API target, entropy expressions, and certificate soundness skeleton.
  • No theorem placeholders are present in the current Lean source.

Why this shape?

This site follows the standard Lean project pattern: a compact landing page whose job is to route readers to the blueprint, generated documentation, dependency graph, and GitHub repository. Since this project is also a reusable library effort, the homepage records the present scope and status explicitly instead of presenting itself as a completed theorem formalization.
