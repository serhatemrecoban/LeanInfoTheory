Project Summary

LeanInfoTheory is a Lean 4/mathlib project building a finite information-theory library and a trusted entropy-inequality certificate checker. The current formalization covers entropy, conditional entropy, mutual information, conditional mutual information, independence and Markov characterizations, finite stochastic-channel data processing, and checked Shannon-inequality demos; the longer-term goal is a textbook-scale library with importable external certificates for network-information-theory converses.

Certificate Validation

The Lean side currently checks certificates; it does not search for them automatically. Given raw certificate data, Lean validates primitive tags, nonnegative rational coefficients, and exact normalized entropy-expression equality.

External tools such as PSITIP or oXitip may later search for certificates. The importer and external tool remain untrusted; Lean validation is the trusted step.
