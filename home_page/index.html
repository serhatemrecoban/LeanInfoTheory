LeanInfoTheory

A Lean/mathlib project for finite information measures, entropy-expression certificates, and future network-information-theory converse automation.

by Serhat Emre Coban

Blueprint (web) Documentation Dependency graph GitHub

LeanInfoTheory is an early-stage Lean/mathlib project for finite information measures and a planned certificate/converse pipeline.

Current milestone: the repository contains a mathlib-based finite-PMF Shannon foundation, primitive Shannon inequalities, and a first proof-carrying checked-certificate layer with exact rational expression matching plus basic raw-to-checked validation. The first non-toy certificate demo proves entropy submodularity from a validated CMI certificate. Semantic KL/conditional-law bridge theorems, external certificate import, generated API documentation, and network-converse examples are planned next milestones.

Implemented now

  • Lake project initialized with Lean 4.30.0 and mathlib 4.30.0.
  • GitHub Actions runs the Lean build and a strict no-placeholder scan.
  • Finite Shannon entropy is defined for mathlib PMFs using Real.negMulLog.
  • Finite entropy is invariant under equivalence and injective alphabet relabelings.
  • Conditional entropy, mutual information, and conditional mutual information are available as entropy-identity definitions.
  • Marginal mass formulas, support/domination lemmas, and coordinate-orientation lemmas are part of the current API.
  • The certificate layer contains formal entropy expressions and a generic nonnegative-combination soundness theorem.
  • An abstract Shannon entropy valuation layer packages empty entropy, conditional entropy nonnegativity, and conditional mutual information nonnegativity assumptions.
  • Primitive Shannon inequality expressions are available for empty entropy, conditional entropy, and conditional mutual information, with soundness theorems under abstract Shannon entropy valuations.
  • A raw/checked certificate split is available; checked certificates use primitive Shannon ingredients, nonnegative rational coefficients, exact rational decomposition equality, and a first raw-to-checked validator.
  • A first non-toy certificate demo proves entropy submodularity from a validated conditional-mutual-information certificate.
  • A separated semantic bridge module exists as an import boundary for future KL-divergence and conditional-law equivalence theorems.
  • No theorem placeholders are present in the current Lean source.

Planned, not yet implemented

  • Semantic bridge theorems identifying the entropy-identity definitions with conditional-law and KL-divergence semantics.
  • Entropy upper bounds by logarithm of alphabet size and the uniform-law equality case.
  • Import support for external certificate formats and richer validation front ends.
  • Generated API documentation; the current documentation page is a module list.
  • Non-toy network-information-theory converse examples.

Why this shape?

This site follows the standard Lean project pattern: a compact landing page whose job is to route readers to the blueprint, current module list, future generated documentation, dependency graph, and GitHub repository. Since this project is also a reusable library effort, the homepage records the present scope and status explicitly instead of presenting itself as a completed theorem formalization.
