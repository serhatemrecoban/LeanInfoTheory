Roadmap

A 3 / 6 / 12 month plan for the library and certificate checker.

This roadmap separates completed Lean/site work from active theorem planning, planned theorem work, and later infrastructure milestones. The labels are intentionally conservative: they describe the current repository, not aspirations.

Status Snapshot

Status Current meaning Examples
Completed Checked Lean work or completed website/documentation work. Project B Chunks 1-3, finite Shannon and semantic/KL bridges, stochastic data processing, four checked certificate demos, and generated reference artifacts.
Active Current near-term theorem planning and maintenance work. Keeping public references aligned with the checked Lean code; no post-Chunk-3 theorem phase is selected yet.
Planned Next technical goals that are not yet formalized or generated. Sufficient statistics, Fano, richer certificate assumptions, and fuller generated documentation.
Later Larger infrastructure or research-facing milestones. PSITIP/oXitip import, network converse skeletons, and a technical report or paper.

Completed

  • Completed Finite PMF-based entropy, conditional entropy, mutual information, and conditional mutual information.
  • Completed Entropy relabeling, zero-entropy characterizations, coordinate reassociation, alphabet/support upper bounds, and uniform-law equality.
  • Completed Project B Chunk 1: pair/triple chain rules, support-aware functional dependence, deterministic processing, MI/CMI identities, inequality bands, reviewed aliases, and opt-in units.
  • Completed Project B Chunk 2: finite KL support and equality, sharp entropy equality cases, ordinary and conditional independence, and zero-MI/zero-CMI characterizations.
  • Completed Project B Chunk 3: finite stochastic channels, Markov characterizations and factorization, MI/KL data processing, invariant-law contraction, and entropy growth under doubly stochastic channels.
  • Completed Semantic bridge theorems for expected self-information, conditional laws, KL divergence, fiber formulas, and nonnegativity.
  • Completed demos Checked certificates for submodularity, subadditivity, monotonicity, and three-way subadditivity.
  • Completed Website dashboard, theorem highlights, module guide, source-derived declaration index, module dependency map, prior-art page, and trusted-flow demo diagram.

Active

  • Unscheduled Select the next theorem phase separately after the completed Chunk 3 checkpoint.
  • Active Keep public references and maintained project notes aligned with checked Lean declarations.

Planned

  • Planned Extend the current module dependency map and source-derived declaration index toward theorem-level leanblueprint pages, a blueprint PDF, and full Lean doc-gen output.
  • Planned Add richer certificate assumptions such as independence, functional dependence, and Markov constraints.
  • Planned Continue the Cover-Thomas Chapter 2 layer with sufficient statistics and Fano in later focused chunks.
  • Planned Prepare focused mathlib PRs for generic, stable finite-measure lemmas.

Later

  • Later Add PSITIP/oXitip-style import into untrusted raw certificate data.
  • Later Formalize selected network-information-theory converse proof skeletons.
  • Later Write a technical report or short paper describing the Lean library and certificate pipeline.
